3.3 Listas e Tabelas

Este resumo aborda as principais técnicas para estilizar listas e tabelas com CSS, mostrando propriedades úteis e exemplos práticos.

3.3.1 Projetando listas

Listas (ordenadas e não ordenadas) podem ser personalizadas através de propriedades como list-style-type, list-style-image e list-style-position. Também é comum controlar margens e recuos para alinhar os marcadores.

Você pode alterar o marcador padrão, usar imagens como marcadores, ou transformar a lista em uma sequência horizontal com CSS sem alterar o HTML.

Exemplo de lista

Exemplo de lista não ordenada com marcador personalizado e uma lista horizontal (navegação):

  • Item A
  • Item B
  • Item C

Lista horizontal (ex.: menu):

  • Home
  • Sobre
  • Contato

/* marcador diferente e posição interna */
ul.custom {
    list-style-type: square;
    list-style-position: inside;
    margin-left: 0;
    padding-left: 0.5em;
}

/* lista horizontal (sem envolver em div) */
ul.horizontal li {
    display: inline-block;
    margin-right: 1em;
}

/* usar números romanos */
ol.web-technologies {
    list-style-type: lower-roman;
}

3.3.2 Projetando tabelas

Tabelas têm propriedades próprias para controlar bordas, espaçamento de células e layout. As principais são border-collapse, border-spacing, table-layout e estilos aplicados a th e td.

Use border-collapse: collapse para unir bordas, defina padding nas células para dar espaço ao conteúdo, e table-layout: fixed quando quiser colunas com larguras previsíveis.

Exemplo de tabela

Abaixo há uma tabela de exemplo que usa a classe demo (veja os estilos em seguida):

Vendas trimestrais
Produto Q1 Q2 Q3 Q4
Caneca 120 135 150 160
Camiseta 200 220 210 230
Boné 75 80 90 95

Estilos CSS usados na tabela:


table.demo {
    width: 100%;
    border-collapse: collapse; /* remove espaçamento entre bordas */
    table-layout: auto; /* ou fixed para larguras fixas */
}

th, td {
    border: 1px solid #ccc;
    padding: 8px;
    text-align: left;
}

caption {
    caption-side: top;
    font-weight: bold;
}

/* zebra striping */
/* zebra striping: aplicar às linhas do corpo da tabela (alternar linhas 1,3,...) */
table.demo tbody tr:nth-child(odd) {
    background-color: #000000;
}

thead tr {
	background-color: #062912;
}

3.3.3 Boas práticas

- Mantenha contraste e legibilidade nas células e marcadores. - Evite usar tabelas para layout; use-as para dados tabulares. - Prefira regras externas (arquivo CSS) para reaproveitamento e manutenção.

Inspecione com as ferramentas do navegador para entender quais regras estão sendo aplicadas e teste o comportamento em diferentes larguras de tela.
