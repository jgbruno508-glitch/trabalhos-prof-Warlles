3.2 Aplicando Cores e Formatação de Texto

Este resumo apresenta as ideias principais do Capítulo 3.2 sobre como aplicar cores, fontes e formatação de texto com CSS.

3.2.1 Cores

O CSS permite definir cores de diversas maneiras: palavras-chave (por exemplo yellow), valores RGB/RGBA, e HSL/HSLA. O canal alfa (no RGBA/HSLA) controla a opacidade e vai de 0.0 até 1.0.

As cores nomeadas são úteis para começar, mas os formatos numéricos dão mais controle. RGB define vermelho, verde e azul; RGBA acrescenta transparência; HSL trabalha com matiz, saturação e luminosidade; e HSLA também inclui transparência.

Exemplos de valores de cor: rgba(255,255,0,0.5), rgb(255, 0, 0), hsl(60,100%,50%) e hsla(60,100%,50%,0.5). Use HSL quando quiser ajustar o tom da cor de forma mais intuitiva. 


h1 {
    color: darkblue;
    background-color: yellow;
}

.destaque {
    color: rgba(255, 0, 0, 0.7);
}

Um uso comum é destacar títulos com fundo colorido e deixar trechos importantes com uma cor mais chamativa. A escolha da cor também deve considerar contraste para facilitar a leitura.

3.2.2 Projetando textos

CSS controla a aparência do texto: família de fontes (font-family), estilo (font-style), peso (font-weight), tamanho (font-size) e espaçamentos entre letras e palavras.

Para fontes, prefira conjuntos alternativos e forneça fallback: por exemplo 'Times New Roman', Times, serif ou Arial, sans-serif. Evite fontes decorativas para textos longos, porque elas cansam a leitura.

O texto também pode ser alinhado, recuado e espaçado com propriedades como text-align, text-indent, line-height, word-spacing e letter-spacing. 


p {
    font-family: Arial, sans-serif;
    text-align: justify;
    text-indent: 50px;
    line-height: 1.4;
}

Esse tipo de configuração deixa o texto mais organizado e legível, principalmente em parágrafos longos e em páginas com muito conteúdo.

3.2.3 Estilo e peso

A propriedade font-style aceita valores como normal, italic e oblique. Já font-weight controla a espessura do texto, podendo usar normal, bold ou valores numéricos como 400 e 700.

Italic é uma fonte inclinada de verdade, enquanto oblique costuma ser uma inclinação simulada. Na prática, os dois servem para dar destaque ao texto, mas o uso deve ser moderado. 


.introducao {
    font-style: italic;
    font-weight: bold;
}

.autor {
    font-style: normal;
    font-weight: 700;
}

Esse recurso é útil para criar ênfase, títulos secundários, legendas e trechos assinados sem precisar alterar a estrutura do HTML.

3.2.4 Tamanhos de fonte

Fontes podem ser especificadas em pixels (px), percentuais (%) ou unidades relativas como em. px é absoluto; em e porcentagens são relativas e úteis para acessibilidade e adaptação de layout.

O valor em em costuma ser baseado no tamanho da fonte do elemento pai. Assim, quando um texto precisa acompanhar o restante da página, unidades relativas ajudam bastante. 


body {
    font-size: 16px;
}

h1 {
    font-size: 2em;
}

p {
    font-size: 100%;
}

Uma boa prática é evitar tamanhos muito pequenos e testar o resultado em diferentes telas para manter a leitura confortável.

3.2.5 Pseudo-classes e pseudo-elementos

O capítulo mostra pseudo-classes úteis para links e seleção de texto (:hover, :active, :focus) e pseudo-elementos como ::first-line e ::selection para estilizar partes específicas do conteúdo.

Pseudo-classes representam estados de um elemento, como quando o mouse passa por cima ou quando um campo recebe foco. Pseudo-elementos, por outro lado, permitem estilizar partes internas do conteúdo, como a primeira linha de um parágrafo.

A propriedade text-transform também aparece no capítulo para alterar a forma visual do texto. Os valores mais usados são:

  • uppercase: transforma todo o texto em letras maiúsculas.
  • lowercase: transforma todo o texto em letras minúsculas.
  • capitalize: deixa a primeira letra de cada palavra em maiúscula.
  • none: mantém o texto como foi escrito no HTML.

a:hover {
    color: red;
}

a:focus {
    color: orange;
}

p::first-line {
    text-transform: uppercase;
}

p.capitalizado {
    text-transform: capitalize;
}

p::selection {
    background-color: yellow;
}

Esses recursos deixam a interface mais interativa e ajudam a destacar ações do usuário e trechos importantes do texto.

3.2.6 Exemplo e boas práticas

Combine regras com cuidado para evitar conflitos: defina uma hierarquia de fontes, prefira arquivos CSS externos para reaproveitamento e use ferramentas de desenvolvedor para inspecionar regras aplicadas.

No capítulo, o exemplo final mostra como o mesmo conteúdo muda visualmente quando recebe estilos de cor, fonte, peso, espaçamento e pseudo-classes. Isso ajuda a perceber que CSS não altera só a aparência, mas também a organização visual da informação. 


/* Exemplo simples */
body { 
    font-family: 'Times New Roman', Times, serif; 
}
h1 { 
    color: green; background-color: yellow; 
}
p { 
    line-height: 1.4; 
}
a:hover { 
    color: red; 
}
